CSS媒体查询

针对不同的媒体类型定义不同的样式

/*最大屏幕分辨率为1280px*/
@media screen and (max-width:1280px){
	.content{ 
		width:20%
	}
}
/*最小屏幕分辨率为1280px且最大为1440px*/
@media (min-width: 1280px) and (max-width:1440px){
	.content{ 
		width:40%
	}
}
/*最小屏幕分辨率为1440px且最大为1680px*/
@media (min-width: 1440px) and (max-width:1680px){
	.content{ 
		width:60%
	}
}
/*最小屏幕分辨率为1920px*/
@media screen and (min-width:1920px){
	.content{ 
		width:100%
	}
}

overflow

overflow: scroll

内容自动撑开容器的宽度不包括滚动条，始终显示滚动条

overflow: auto

内容撑开容器的宽度包括滚动条，按需显示滚动条，带来的滚动条会占用内容的一部分宽度，导致内容显示不全
