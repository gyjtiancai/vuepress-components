😕 vue

安装vue

npm install vue

  • vue官网下载地址

安装vue-cli

npm install -g @vue/cli

  • vue-cli官网下载地址

检查vue-cli是否安装成功

vue --version
1

接下来便可以快速创建vue项目

vue create my-project
1

创建成功后若想用typescript写vue项目

vue add typescript
1

windows 且使用 Git Bash 的小伙伴一定要看这里！！！

引用官方文档的话：
如果你在 Windows 上使用 Git Bash，交互提示符并不工作。你必须通过 winpty vue.cmd create my-project 启动这个命令。不过，如果你仍想使用 vue create hello-world，则可以通过在 ~/.bashrc 文件中添加以下行来为命令添加别名。 alias vue='winpty vue.cmd' 你需要重新启动 Git Bash 终端会话以使更新后的 bashrc 文件生效。

简单的总结就是：

windows中 使用vue create my-project 指令 快速创建项目 在Git Bash直接使用是会提示指令不存在的 而在cmd直接可以使用

如果一定要在Git Bash中使用

  • 方法一：使用winpty vue.cmd create my-project指令
  • 方法二：找到C:\Program Files\Git\etc文件夹下的bash.bashrc文件，用记事本打开并且在末尾设置别名alias vue='winpty vue.cmd' 便可以使用vue create my-project了

ok 搞定 😑

启动项目

npm run serve
