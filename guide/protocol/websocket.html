websocket协议 😑

概念

  • WebSocket是HTML5新增的协议，它的目的是在浏览器和服务器之间建立一个不受限的双向通信的通道，比如说，服务器可以在任意时刻发送消息给浏览器
  • WebSocket并不是全新的协议，而是利用了HTTP协议来建立连接

对比http协议

为什么传统的HTTP协议不能做到WebSocket实现的功能？这是因为HTTP协议是一个请求－响应协议，请求必须先由浏览器发给服务器，服务器才能响应这个请求，再把数据发送给浏览器。换句话说，浏览器不主动请求，服务器是没法主动发数据给浏览器的。
除非js写个定时器，然后以固定的间隔给服务器发请求，询问服务器有没有新消息
但是这个机制的缺点一是实时性不够，二是频繁的请求会给服务器带来极大的压力。
为什么WebSocket连接可以实现全双工通信而HTTP连接不行呢？实际上HTTP协议是建立在TCP协议之上的，TCP协议本身就实现了全双工通信，但是HTTP协议的请求－应答机制限制了全双工通信。WebSocket连接建立以后，就是规定接下来，通信就不使用HTTP协议了，直接互相发数据。

使用websocket

server端（nodejs）

  • 1.安装ws模块 npm i ws

TIP

安装ws之前需要先安装2个插件
1.bufferutil(ws缓冲工具)npm install bufferutil --save-optional
2.utf-8-validate(检查缓冲区是否包含有效的UTF-8编码文本)npm install utf-8-validate --save-optional

  • 2.创建websocket服务
const WebSocket = require('ws');// 导入WebSocket模块:
// console.log('WebSocket',WebSocket)
const WebSocketServer = WebSocket.Server;// 引用Server类:
// console.log('WebSocketServer',WebSocketServer)
// 实例化:
const wss = new WebSocketServer({
    port: 3000
});

wss.on('connection', function (ws) {    
    console.log(`服务端连接成功`);
    ws.on('message', function (message) {
        console.log(`服务端接受到客户端发送的消息: ${message}`);
        ws.send(`服务端接受到了你的消息: ${message}`, (err) => {
            if (err) {
                console.log(`服务端 error: ${err}`);
            }
        });
    })
    ws.on('close', function (message) {
        console.log('连接关闭:', message);
    })
});

  • 2.demo-js
export default {
  name: "Order",
  data() {
    return {
      content: "",
      ws: null,
      messageList: [],
    };
  },
  methods: {
    //初始化Websocket
    initWebsocket() {
      //和服务端建立连接
      this.ws = new WebSocket("ws://localhost:3000");
      // 打开WebSocket
      console.log(this.ws);
      this.ws.onopen = () => {
        this.$message.success("WebSocket打开成功");
        //   ws.send("Hello! 我是客户端");
      };
      // 响应服务端收到的消息:
      this.ws.onmessage = (message) => {
        console.log("客户端收到了服务端的消息:", message);
        this.messageList.push(message.data);
      };
      this.ws.onclose = (message) => {
        console.log("WebSocket关闭:", message);
        console.log(this.ws);
        this.ws = null;
      };
      this.ws.onerror = (message) => {
        console.log("WebSocket出错:", message);
        this.ws = null;
      };
    },
    //发送消息
    sendMessage() {
      if (!this.ws) {
        this.$message.error("请先打开WebSocket");
        return
      }
      this.ws.send(this.content);
      this.content = "";
    },
    //打开
    openWebSocket() {
      this.initWebsocket();
    },
    //关闭WebSocket
    closeWebSocket() {
      this.ws.close();
    },
  },
};
